# portfolio-evidence

보고서(Report) 및 증빙자료(Evidence: 수상 상장 사본, 교육 이수증 등)를 **포트폴리오 관점에서 검증 가능하게 정리**하는 저장소입니다.

---

## Contents

- **Reports**
  - DFIR 분석 보고서, 악성코드 분석 보고서, 프로젝트 산출물 PDF 등
- **Evidence**
  - 수상 상장 사본(마스킹본)
  - 교육/훈련 이수증(마스킹본)
  - 발표/세미나/프로그램 참여 증빙(필요 시)

📌 전체 목록은 `index.md`에서 확인할 수 있습니다.

---

## Privacy & Redaction Policy (중요)

이 레포는 **공개 저장소 기준**으로 운영합니다.  
따라서 아래 항목이 포함된 문서는 **반드시 마스킹(redacted)** 후 업로드합니다.

- 주민등록번호/생년월일/학번/수료번호/식별번호
- 주소/전화번호/이메일
- QR/검증코드/인증 URL(검증 토큰 포함 가능)
- 서명/도장(필요 시)

> 원본(무마스킹) 문서는 **업로드하지 않습니다.**  
> 검증이 필요한 경우 요청 시 별도 채널로 제공할 수 있습니다.

---

## Repository Structure
- `index.md` : 문서 목록(발급기관/날짜/요약/관련 링크)
- `reports/` : 보고서/산출물
- `evidence/` : 상장/이수증/참여 증빙
- `evidence/awards/` : 수상 관련 증빙
- `evidence/certificates` : 자격/인증/공식 수료 증빙
- `evidence/training/` : 교육/훈련/프로그램 이수 증빙
- `evidence/participation/` : 참여/활동/선발/발표/기여 증빙
- `media/` : README/인덱스에서 사용하는 스크린샷(선택)
